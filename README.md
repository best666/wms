# wms
这是一个基于vue的仓库管理demo，属于vue的练手项目，计划实现：
出入调库、仓库/货物/用户管理（增删改查）、报表（考虑）

后续实际使用会使用uniapp开发。

## Project setup
安装依赖：
```
npm install
```

### Compiles and hot-reloads for development
运行项目：
```
npm run serve
```

### Compiles and minifies for production
打包项目：
```
npm run build
```
