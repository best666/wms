# wms
这是一个仓库管理的小demo，计划实现：
出入调库、仓库/货物/用户管理（增删改查）、报表（考虑）

## Project setup
安装依赖：
```
npm install
```

### Compiles and hot-reloads for development
运行项目：
```
npm run serve
```

### Compiles and minifies for production
打包项目：
```
npm run build
```
